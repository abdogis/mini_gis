

 Web Map SIG - Leaflet
Une application web SIG (Système d'Information Géographique) développée avec Leaflet.js permettant
d'importer, visualiser et gérer des données géographiques au format GeoJSON et Shapefile (.zip)
directement dans le navigateur.
## ✨ Fonctionnalités
 Import de fichiers GeoJSON, JSON et Shapefile (.zip)
️ Affichage des données sur une carte interactive Leaflet
 Choix du fond de carte :
OpenStreetMap
## Esri Satellite
 Personnalisation du style des couches :
Couleur de remplissage
Couleur du contour
️ Afficher / masquer les couches
 Réorganiser l'ordre des couches
️ Supprimer une couche
 Consultation des attributs via des popups
 Légende dynamique mise à jour automatiquement
 Interface responsive avec menu latéral escamotable

## ️ Utilisation
- Importer une couche
Cliquez sur "Sélectionner un fichier" puis choisissez :
## .geojson
## .json
.zip (Shapefile)
- Modifier le style
Sélectionnez une couche dans la liste puis choisissez :
couleur du remplissage
couleur du contour
Les modifications sont appliquées instantanément.
- Gérer les couches
Pour chaque couche, il est possible de :
afficher/masquer
déplacer vers le haut
déplacer vers le bas
supprimer
- Consulter les attributs
Cliquez sur un objet de la carte pour afficher sa table d'attributs.
